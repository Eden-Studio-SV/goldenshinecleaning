# Golden Shine Cleaning Service — Website

Bilingual (EN / ES) marketing homepage for Golden Shine Cleaning Service — sticky
header with a language toggle, hero, services, how-it-works, reviews, an
interactive quote form, and footer.

## What's here

- **`index.html`** — the live site. A single self-contained static page: all
  JavaScript, fonts (Great Vibes / Poppins / Mulish), and artwork are inlined.
  No build step, no external CDN or network dependency.
- **`golden-shine-design-system/`** — the Claude Design handoff this page was
  built from (brand tokens, component sources, and the UI-kit source).

## Deploy (Vercel)

Zero-config static deploy. Vercel serves `index.html` at the root — no build
command and no output directory are required.

- **Framework Preset:** Other
- **Build Command:** _(none)_
- **Output Directory:** _(root)_
