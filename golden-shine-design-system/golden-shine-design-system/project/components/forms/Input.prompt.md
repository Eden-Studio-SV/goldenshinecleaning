Labeled text input used across quote forms and contact fields. Coral focus ring matches the brand.

```jsx
<Input label="Full name" placeholder="Jane Doe" />
<Input label="Email" type="email" leftIcon={<MailIcon />} helper="We'll send your quote here" />
<Input label="Phone" error="Please enter a valid number" />
```

Supports `label`, `helper`, `error` (red state), `leftIcon`, `disabled`. Field height is 50px with a 14px radius.
