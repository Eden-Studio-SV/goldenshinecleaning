Circular icon-only button — for compact actions like phone, WhatsApp, menu, or social links. Always pass an accessible `label`.

```jsx
<IconButton label="Call us" variant="accent"><PhoneIcon /></IconButton>
<IconButton label="Menu" variant="soft"><MenuIcon /></IconButton>
```

Variants: `solid` (navy), `accent` (coral), `soft` (pale navy tint), `ghost` (outline). Sizes `sm`/`md`/`lg` map to 36/44/52px — never go below 44px for primary touch targets.
